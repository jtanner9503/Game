  • Choose a class. Fighters have high health and always block the first hit, wizards have high damage and always land the first hit, and rogues sneak and open chests.
  • If you are class 1 or 2, the goal is to defeat the dragon. Otherwise, get 22000 gold.
  • Start in Great Hall.
  • Roll dice. Level up if you land on L. If you are class 3, open chests that you land on.
  • If you go into a room as class 1 or 2, set enemy level to the number indicated and fight the enemy. Level up afterwards. If you are class 3, then sneak. If you fail, fight the enemy. Rogues get 2000 gold afterwards.
  • If you are a level 10 Fighter or wizard, go back to the Great Hall and fight the dragon.
